# Retail Sales Dashboard using Power BI

## Overview

This project analyzes retail sales performance using Power BI.

## Tools Used

- Microsoft Excel
- Power BI Desktop
- DAX
- Power Query

## Dashboard Features

- KPI Cards
- Sales by Region
- Sales by Category
- Sales Trend
- Top Products
- Customer Segment Analysis
- Interactive Slicers

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Total Quantity
- Profit Margin

## Dataset

- Orders
- Customers
- Product
- Returns
