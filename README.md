# Retail Sales Dashboard

## Overview

This project is an interactive Retail Sales Dashboard built using Power BI.

It provides insights into:

- Total Sales
- Total Profit
- Total Orders
- Customer Analysis
- Regional Performance
- Product Category Analysis
