# 📊 Retail Sales Dashboard

## Project Overview

This project is an interactive Retail Sales Dashboard developed using Microsoft Power BI.

It helps analyze:

- Sales Performance
- Profit Analysis
- Customer Analysis
- Regional Sales
- Product Category Performance
